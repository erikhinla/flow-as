Key Components & Agents

BizBrain Lite FastAPI service on port 18000. Accepts task envelopes, validates schema, writes to Postgres, enqueues to Redis.
OpenClaw Routing agent. Reads task_type + risk_tier to decide which agent owns the job.
Hermes Content & classification agent. Handles content_prep, rewrite, skill_extraction tasks.
Agent Zero Implementation agent. Handles complex builds, landing pages, campaigns, multi-step deliverables.
Redis Queue Durable FIFO per owner. Keys: flow:{owner}:jobs. Workers use BRPOP.
Postgres job_records table tracks full status lifecycle. reflection_records store learning.
OpenRouter LLM gateway. Routes to GPT-4o, Claude, Mistral based on task complexity and cost tier.
runtime/reviews/ Output artifact directory. Every completed job writes output.md and metadata.json here.
