# FLOW Agent AS Visual Asset Reference Drop

Use this folder for approved source/reference assets only.

Do not place assets in legacy SaaS, old FLOW Agent OS, or duplicate folders.

## Folders

- `logos/` - official logos, marks, wordmarks, icon files
- `screenshots/` - product, site, dashboard, proof-of-work screenshots
- `founder/` - founder photos and bio/story visuals
- `product-mockups/` - offer previews, report mockups, kit previews
- `brand-inspiration/` - references for style direction, mood, layout, typography
- `raw/` - unsorted incoming assets to classify later

## Naming

Prefer descriptive names:

- `transformby10x-logo-primary.png`
- `bizbuilders-ava-intake-screen-2026-05.png`
- `prompting-circumstance-proof-passenger-doesnt-speak.png`
- `revanew-product-mockup-v1.png`

Avoid names like:

- `final.png`
- `new logo.png`
- `screenshot 1.png`
- `use this maybe.png`
