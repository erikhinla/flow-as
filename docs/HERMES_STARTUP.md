# Hermes Startup

## Startup objective
Initialize Hermes as a bounded execution agent inside FLOW Agent OS.

## Preconditions
- Canon exists in PROJECTS/_OS/02-Canon/
- FLOW_AGENT_OS/docs/_authority.md exists
- Execution Engine positioning is already defined
- The requested task has a clear WIN

## Startup steps
1. Read:
   - PROJECTS/_OS/02-Canon/CANONICAL_SOURCE_OF_TRUTH.md
   - PROJECTS/_OS/02-Canon/NAMING_GLOSSARY.md
   - PROJECTS/_OS/02-Canon/ANTI_DRIFT_RULES.md
   - PROJECTS/FLOW_AGENT_OS/docs/_authority.md
   - PROJECTS/FLOW_AGENT_OS/docs/EXECUTION_ENGINE_OVERVIEW.md

2. Confirm:
   - Hermes is inside FLOW Agent OS
   - Hermes is not doctrine
   - Hermes is not activation
   - Hermes is not interface

3. Load task:
   - WIN statement
   - target surface
   - required output
   - definition of done

4. Check risk:
   - low-risk: continue
   - high-risk: escalate

5. Execute bounded task

6. Write output to the correct destination

7. Record:
   - what changed
   - what was produced
   - what was blocked
   - whether escalation is required

## Startup failure conditions
- Canon unavailable
- WIN unclear
- Target output undefined
- Risk exceeds allowed scope
